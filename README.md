# 👤 Verificador de Idade

Projeto desenvolvido durante meus estudos de JavaScript com o objetivo de praticar estruturas condicionais, validação de dados e manipulação do DOM.

## 🚀 Funcionalidades

- Calcula a idade do usuário
- Identifica o sexo informado
- Exibe uma imagem correspondente à faixa etária
- Valida campos obrigatórios

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript

## 📚 Conceitos praticados

- Manipulação do DOM
- Estruturas condicionais (`if` / `else`)
- Operadores lógicos
- Objeto `Date`
- Validação de formulários
- Manipulação dinâmica de imagens
